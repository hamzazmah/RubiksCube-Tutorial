# RubiksCube-Tutorial
A simple Rubik's Cube Simulator made using Processing. Tutorial by "The Coding Train".

### Controls

Hold Left Click and move mouse to rotate the Cube.

#### Moves

f - Front
b - Back
u - Up
d - Down
l - Left
r - Right
